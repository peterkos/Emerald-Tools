#Emeralds+
######Putting some use to Emeralds

This mod allows you to create items with Emeralds, such as

- Emerald Pickaxe
- Emerald Sword
- Emerald Shovel
- Emerald Axe
- Emerald Hoe

Also, there's armor.

- Eemerald Helmet
- Emerald Chestplate
- Emerald Leggings
- Emerald Boots


Finally: Achievements!

####We need to get greener! `Acquire Emeralds`

####High Miner! `Craft an Emerald Pickaxe`
